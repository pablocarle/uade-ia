Instalacion y ejecucion del aplicativo

El sistema debe tener instalado Java en su version 8 o superior.

Se debe instalar la libreria CLIPSJNI, siguiendo las instrucciones del proveedor:

http://clipsrules.sourceforge.net/CLIPSJNIBeta.html

https://sourceforge.net/projects/clipsrules/files/CLIPS/6.30/clips_jni_050.zip/download
